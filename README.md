# AI-Complaint-Analyzer
AI-powered tool for analyzing e-commerce customer complaints using a local LLM (Ollama). Generates summary, category, severity, sentiment, and suggested response.


## Features & Functionality

- Input: Customer complaint (text)

- Output:
  - Summary (short 1–2 lines)
  - Severity (Low / Medium / High)
  - Category (Delivery / Product / Payment / Service)
  - Sentiment (Positive / Neutral / Negative)
  - Suggested response (professional reply)

---

## ⚙️ Expected Output Format

```json
{
  "summary": "",
  "severity": "",
  "category": "",
  "sentiment": "",
  "response": ""
}
